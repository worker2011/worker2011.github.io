隐私政策

第壹条 本站非常重视用户的隐私和个人信息保护。您在使用网站时，可能会收集和使用您的相关信息。本政策旨在向您说明在您访问 leo2011.eu.org 网站时，如何收集、使用、保存、共享和转让这些信息。

第贰条 下面表格中是搭建本站全部直接服务提供商，大部分为认证的国际大厂或开源项目，隐私性可以保证：

名称 官网 服务内容 与您隐私的关系
Hexo https://hexo.io 提供本站的基础后端框架 不会收集您的任何数据
安知鱼主题 (Hexo-Theme-Anzhiyu) https://docs.anheyu.com 提供本站的前端主题 不会收集您的任何数据
eu-org https://nic.eu.org 提供网站域名 不会收集您的任何数据
GitHub https://github.com 托管本站的网页内容，进行初级部署 会处理您的数据
Cloudflare https://cloudflare-cn.com 配置本站的 DNS 服务和反攻击系统、提供一个电子邮件服务 会处理您的数据
Netlify https://netlify.com 将 Github 的网页进行二次部署到亚洲的 CDN 服务器上，加快访问速度；同时，评论系统的后端也在这里运行 会处理与您评论相关的数据
Let's Encrypt https://letsencrypt.org 提供 SSL 证书，让您用上更安全的 HTTPS 链接 不会收集您的任何数据
MongoDB Atlas https://mongodb.com 存储评论系统的后端数据 会存储与您评论相关的数据，包括 IP 地址、电子邮件等
Twikoo https://twikoo.js.org 提供评论服务、获取 IP 地址并统计文章阅读量 会存储并处理与您评论相关的数据，包括 IP 地址、电子邮件等

第三方服务的共享、转让情况详见对应服务的隐私协议。

第叁条 受资金等情况所限，您的数据将不得不在海外的服务器中处理、存储。

但是本站会尽量选择安全的服务，您也可以通过使用匿名评论功能来保护隐私。

第肆条 在评论的时候，Twikoo、MongoDB Atlas 和 Netlify 会收集或处理您的数据。

您可以选择不提供电子邮件来保护隐私，即使提供了电子邮件也不会公开。您可以通过 VPN 代理访问来保护自己的 IP 地址。

鼓励你使用不易被人恶意识别的昵称进行评论，但是建议你填写真实的邮箱以便收到回复（邮箱信息不会被公开）。

在你评论时，会额外收集你的个人信息。

评论时会记录你的 QQ 帐号（如果在邮箱位置填写 QQ 邮箱或 QQ 号），方便获取你的 QQ 头像。如果使用 QQ 邮箱但不想展示 QQ 头像，可以填写不含 QQ 号的 QQ 邮箱。（主动，存储）

评论时会记录你的邮箱，当我回复后会通过邮件通知你（主动，存储，不会公开邮箱）

评论时会记录你的网址，用于点击头像时快速进入你的网站（主动，存储）

评论时会记录你的 IP 地址，作为反垃圾的用户判别依据（被动，存储，不会公开 IP，会公开 IP 所在省份或同级别行政区）

评论会记录你的浏览器代理，用作展示系统版本、浏览器版本方便展示你使用的设备，快速定位问题（被动，存储）

在评论时，本站内置的第三方服务会通过以下或更多途径，来获取你的相关信息：

cravatar 会收集你的访问信息、评论填写的个人信息用于展示头像

第伍条 本站没有账户注册系统，不会需要您的密码等信息。

您可以自愿填写真实的、或者虚假的信息作为你评论的展示信息。

第陆条 本站及本站使用的服务会处理包括但不限于您的以下信息

  • IP 地址
  • 浏览器 UA
  • 网络身份标识信息
  • 设备信息
  • 浏览过程

第柒条 在访问时，小 L 原则上不会浏览、使用或共享你的任何个人信息，但以下情况除外：

  1. 恶意访问识别，用于维护网站
  2. 恶意攻击排查，用于维护网站
  3. 反垃圾、广告评论、“键盘侠”“喷子” 等不良行为

第捌条 本站为实现无账号评论、深色模式切换等功能，会在你的浏览器中进行本地存储，你可以随时清除浏览器中保存的所有 Cookies 以及 LocalStorage，不影响你的正常使用。

第玖条 本博客中的以下业务会在你的计算机上主动存储数据：

  • 内置服务
  • 评论系统
  • 即刻短文
  • 鱼塘
  • 中控台
  • 胶囊音乐
  • 第三方服务

关于如何使用你的 Cookies，请访问 Cookies 政策。

关于如何在 Chrome 中清除、启用和管理 Cookie。

第拾条 本站没有接入任何广告等营收业务。

第拾壹条 本政策自 年 月 日正式施行。

后续有关任何变动都会及时更新。
